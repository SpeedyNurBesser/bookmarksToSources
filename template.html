Usage

Use %index, %title and %url in your template. They will be replaced on generation with an index value, the bookmark's title or the bookmark's url respectively.

Use html-elements like < i >, < b > or < u >, but note that they will only be rendered, when the file is exported as an .html file.

Note, that formatting like bold, italic or underlined text is currently not supported for exporting as .docx files.
